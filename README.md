## 启动demo项目

1. cd `car-tracking-2d/demos/react-demo`

2. `yarn`

3. `yarn start`

## 界面使用

### debug 模式

浏览器url ?后面(search部分)加入参数`debug=1`

例如：`http://localhost:3000/home?tunnelNo=tunnel1&debug=1`
